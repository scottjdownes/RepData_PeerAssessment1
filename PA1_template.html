What is mean total number of steps taken per day?

Now that we have the data, we’re going to do some calculations.

First, I’ll sum up the number of steps for each day. Then we’ll look at the mean and the median of that data set. After that, we’ll plot the total steps per day by summing steps over our data set, split by day.

stepsPerDay <- tapply(fitnessData$steps, fitnessData$date, FUN=sum)
mean <- mean(stepsPerDay, na.rm = TRUE)
mean
## [1] 10766.19
median <- median(stepsPerDay, na.rm = TRUE)
median
## [1] 10765
plot(stepsPerDay)

What is the average daily activity pattern?

OK, now we want to look at a normal distribution of activity over the course of a day. We do that by splitting on interval and then looking at that data. First, we plot it in a line graph, and then we look for the interval with the max mean steps.

stepsPerInterval <- tapply(fitnessData$steps, fitnessData$interval, FUN=mean, na.rm=TRUE)
plot(stepsPerInterval, type = "l")

maxMeanInterval <- names(stepsPerInterval)[stepsPerInterval == max(stepsPerInterval)]
maxMeanInterval
## [1] "835"

Are there differences in activity patterns between weekdays and weekends?

For this case, we’ll look at the new data set and we’ll need a new function to take care of determining whether the date is a weekend or weekday.

dayChecker <- function(x) {
  dayType <- "Weekday"
  dayName <- weekdays(as.Date(x))
  if (dayName == "Saturday") { dayType <- "Weekend" }
  else if (dayName == "Sunday") { dayType <- "Weekend" }
  dayType
}

stepsPerWeekday <- tapply(fitnessDataZeros$steps, weekdays(as.Date(fitnessDataZeros$date)), FUN=sum)
mean <- mean(stepsPerWeekday, na.rm = TRUE)
mean
## [1] 81515.43
